Detailed Description

template<typename PointT>
class pcl::visualization::PointCloudGeometryHandlerSurfaceNormal< PointT >

Surface normal handler class for PointCloud geometry.

Given an input dataset, all data present in fields "normal_x", "normal_y", and "normal_z" is extracted and displayed on screen as XYZ data.
