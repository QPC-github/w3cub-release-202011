Syntax function bind

Syntax only

This plugin only allows Babel to parse this syntax. If you want to transform it then see transform-function-bind.

This plugin allows Babel to parse function bind syntax.

Installation

npm install --save-dev babel-plugin-syntax-function-bind

Usage

Via .babelrc (Recommended)

.babelrc

{
  "plugins": ["syntax-function-bind"]
}

Via CLI

babel --plugins syntax-function-bind script.js

Via Node API

require("babel-core").transform("code", {
  plugins: ["syntax-function-bind"]
});
