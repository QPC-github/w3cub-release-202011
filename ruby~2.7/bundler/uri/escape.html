def escape(*arg)
  warn "Bundler::URI.escape is obsolete", uplevel: 1
  DEFAULT_PARSER.escape(*arg)
end

Synopsis

Bundler::URI.escape(str [, unsafe])

Args

str

String to replaces in.

unsafe

Regexp that matches all symbols that must be replaced with codes. By default uses UNSAFE. When this argument is a String, it represents a character set.

Description

Escapes the string, replacing all unsafe characters with codes.

This method is obsolete and should not be used. Instead, use CGI.escape, Bundler::URI.encode_www_form or Bundler::URI.encode_www_form_component depending on your specific use case.

Usage

require 'bundler/vendor/uri/lib/uri'

enc_uri = Bundler::URI.escape("http://example.com/?a=\11\15")
# => "http://example.com/?a=%09%0D"

Bundler::URI.unescape(enc_uri)
# => "http://example.com/?a=\t\r"

Bundler::URI.escape("@?@!", "!?")
# => "@%3F@%21"
