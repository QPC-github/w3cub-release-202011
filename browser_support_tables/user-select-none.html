Notes

* Partial support with prefix.

Bugs

  • iOS does not allow input elements to be focused (and thus prevents text input) when the element has -webkit-user-select: none set

  • Reported to not work in some manufacturer's versions of Android 4.0 and below, though confirmed to work in others.
