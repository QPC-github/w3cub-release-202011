Notes

  1. Partial support in older Safari versions refers to not working in HTML files or CSS background images.

Bugs

  • No events are fired at all in WebKit/Blink browsers during animation (onbegin/onrepeat/onend) see bug

  • Animation in SVG is not supported in inline SVG on Android browsers prior to version 4.4.
