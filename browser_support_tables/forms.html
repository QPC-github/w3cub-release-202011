Notes

Bugs

  • Safari (>= 5.1) on Windows has a bug in multiple file uploads when used together with FormData to do binary uploads; each file will be reported as 0 bytes. Single file selection, however, works fine.

  • File uploading is not possible on iOS Safari before iOS 6.
