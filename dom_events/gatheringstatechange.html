gatheringstatechange

A gatheringstatechange event occurs when the RTCIceTransport's gathering state changes. The gathering state, whose actual status can be found in the transport object's gatheringState property, indicates whether or not the ICE agent has begun gathering candidates, and if so, if it has finished doing so.
