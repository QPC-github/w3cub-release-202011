public safeStringifyJson(value: any): string source

Stringify a value as JSON with some differences:

  • bigints are stringified as a json string. (safeStringifyJson({ val: 1n }) outputs '{ "val": "1" }'). This is because of a decision by TC39 to not support bigint in JSON.stringify https://github.com/tc39/proposal-bigint/issues/24

Params:

Name Type Attribute Description
value any

the value to stringify.

Return:

string

the resulting json.
