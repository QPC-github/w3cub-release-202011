staticCFunction

kotlin-stdlib / kotlinx.cinterop / staticCFunction
Platform and version requirements: Native (1.3) 
fun <R> staticCFunction(
    function: () -> R
): CPointer<CFunction<() -> R>>

Returns a pointer to C function which calls given Kotlin static function.

Parameters

function - must be static, i.e. an (unbound) reference to a Kotlin function or a closure which doesn't capture any variable
