zip_entry_compressionmethod

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_entry_compressionmethod — Retrieve the compression method of a directory entry

Description

zip_entry_compressionmethod ( resource $zip_entry ) : string

Returns the compression method of the directory entry specified by zip_entry.

Parameters

zip_entry

A directory entry returned by zip_read().

Return Values

The compression method.

Changelog

Version Description
8.0.0 This function is deprecated in favor of the Object API, see ZipArchive::statIndex().
