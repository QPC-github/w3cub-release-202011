All Implemented Interfaces:
Serializable
public class LinkLoopException
extends LinkException

This exception is thrown when a loop was detected while attempting to resolve a link, or an implementation specific limit on link counts has been reached.

Synchronization and serialization issues that apply to LinkException apply directly here.

1.3
