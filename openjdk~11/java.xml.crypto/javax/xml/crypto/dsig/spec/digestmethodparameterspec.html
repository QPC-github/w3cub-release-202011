Interface DigestMethodParameterSpec

All Superinterfaces:
AlgorithmParameterSpec
public interface DigestMethodParameterSpec
extends AlgorithmParameterSpec

A specification of algorithm parameters for a DigestMethod algorithm. The purpose of this interface is to group (and provide type safety for) all digest method parameter specifications. All digest method parameter specifications must implement this interface.
