Package javax.xml.namespace

Defines XML Namespace processing.

The following XML standards apply:

  • XML Schema Part2: Datatypes specification
  • Namespaces in XML
  • Namespaces in XML Errata
Since:
1.5
Interface Summary
Interface Description
NamespaceContext

Interface for read only XML Namespace context processing.
Class Summary
Class Description
QName

QName represents a qualified name as defined in the XML specifications: XML Schema Part2: Datatypes specification, Namespaces in XML.
