Qt State Machine

The State Machine framework provides classes for creating and executing state graphs.

Getting Started

To include the definitions of the module's classes, use the following directive:

#include <QStateMachine>

To import the QML types into your application, use the following import statement in your .qml file:

import QtQml.StateMachine

To link against the module:

When using CMake add to CMakeLists.txt:

find_package(Qt6 COMPONENTS StateMachine REQUIRED)
target_link_libraries(mytarget PRIVATE Qt6::StateMachine)

When using qmake add to the project .pro file:

QT += statemachine
