CQueueIterator

Package system.collections
Inheritance class CQueueIterator
Implements Iterator, Traversable
Since 1.0
Source Code framework/collections/CQueueIterator.php
CQueueIterator implements an iterator for CQueue.

It allows CQueue to return a new iterator for traversing the items in the queue.
