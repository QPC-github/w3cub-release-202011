arguments[@@iterator]()

The @@iterator method of the arguments object implements the iterable protocol and allows arguments to be consumed by most syntaxes expecting iterables, such as the spread syntax and for...of loops.

The initial value of this property is the same function object as the initial value of the Array.prototype.values property (and also the same as Array.prototype[@@iterator]).

Property attributes of arguments[@@iterator]()
Writable yes
Enumerable no
Configurable yes

Syntax

arguments[Symbol.iterator]()

Examples

Iteration using for...of loop

function f() {
  for (const letter of arguments) {
    console.log(letter);
  }
}
f('w', 'y', 'k', 'o', 'p');
