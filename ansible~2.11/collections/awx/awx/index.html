Awx.Awx

Collection version 17.0.1

Plugin Index

These are the plugins in the awx.awx collection

Inventory Plugins

  • tower – Ansible dynamic inventory plugin for Ansible Tower.

Lookup Plugins

  • tower_api – Search the API for objects
  • tower_schedule_rrule – Generate an rrule string which can be used for Tower Schedules

Modules

  • tower_ad_hoc_command – create, update, or destroy Ansible Tower ad hoc commands.
  • tower_ad_hoc_command_cancel – Cancel an Ansible Tower Ad Hoc Command.
  • tower_ad_hoc_command_wait – Wait for Ansible Tower Ad Hoc Command to finish.
  • tower_application – create, update, or destroy Ansible Tower applications
  • tower_credential – create, update, or destroy Ansible Tower credential.
  • tower_credential_input_source – create, update, or destroy Ansible Tower credential input sources.
  • tower_credential_type – Create, update, or destroy custom Ansible Tower credential type.
  • tower_export – export resources from Ansible Tower.
  • tower_group – create, update, or destroy Ansible Tower group.
  • tower_host – create, update, or destroy Ansible Tower host.
  • tower_import – import resources into Ansible Tower.
  • tower_instance_group – create, update, or destroy Ansible Tower instance groups.
  • tower_inventory – create, update, or destroy Ansible Tower inventory.
  • tower_inventory_source – create, update, or destroy Ansible Tower inventory source.
  • tower_inventory_source_update – Update inventory source(s).
  • tower_job_cancel – Cancel an Ansible Tower Job.
  • tower_job_launch – Launch an Ansible Job.
  • tower_job_list – List Ansible Tower jobs.
  • tower_job_template – create, update, or destroy Ansible Tower job templates.
  • tower_job_wait – Wait for Ansible Tower job to finish.
  • tower_label – create, update, or destroy Ansible Tower labels.
  • tower_license – Set the license for Ansible Tower
  • tower_meta – Returns metadata about the collection this module lives in.
  • tower_notification_template – create, update, or destroy Ansible Tower notification.
  • tower_organization – create, update, or destroy Ansible Tower organizations
  • tower_project – create, update, or destroy Ansible Tower projects
  • tower_project_update – Update a Project in Ansible Tower
  • tower_receive – Receive assets from Ansible Tower.
  • tower_role – grant or revoke an Ansible Tower role.
  • tower_schedule – create, update, or destroy Ansible Tower schedules.
  • tower_send – Send assets to Ansible Tower.
  • tower_settings – Modify Ansible Tower settings.
  • tower_team – create, update, or destroy Ansible Tower team.
  • tower_token – create, update, or destroy Ansible Tower tokens.
  • tower_user – create, update, or destroy Ansible Tower users.
  • tower_workflow_job_template – create, update, or destroy Ansible Tower workflow job templates.
  • tower_workflow_job_template_node – create, update, or destroy Ansible Tower workflow job template nodes.
  • tower_workflow_launch – Run a workflow in Ansible Tower
  • tower_workflow_template – create, update, or destroy Ansible Tower workflow template.
