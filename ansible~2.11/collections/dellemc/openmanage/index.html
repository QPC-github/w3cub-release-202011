Dellemc.Openmanage

Collection version 3.0.0

Plugin Index

These are the plugins in the dellemc.openmanage collection

Modules

  • dellemc_configure_idrac_eventing – Configures the iDRAC eventing related attributes
  • dellemc_configure_idrac_services – Configures the iDRAC services related attributes
  • dellemc_get_firmware_inventory – Get Firmware Inventory
  • dellemc_get_system_inventory – Get the PowerEdge Server System Inventory
  • dellemc_idrac_lc_attributes – Enable or disable Collect System Inventory on Restart (CSIOR) property for all iDRAC/LC jobs
  • dellemc_idrac_storage_volume – Configures the RAID configuration attributes
  • dellemc_system_lockdown_mode – Configures system lockdown mode for iDRAC
  • idrac_bios – Configure the BIOS attributes
  • idrac_firmware – Firmware update from a repository on a network share (CIFS, NFS, HTTP, HTTPS, FTP)
  • idrac_firmware_info – Get Firmware Inventory
  • idrac_lifecycle_controller_job_status_info – Get the status of a Lifecycle Controller job
  • idrac_lifecycle_controller_jobs – Delete the Lifecycle Controller Jobs
  • idrac_lifecycle_controller_logs – Export Lifecycle Controller logs to a network share
  • idrac_lifecycle_controller_status_info – Get the status of the Lifecycle Controller
  • idrac_network – Configures the iDRAC network attributes
  • idrac_os_deployment – Boot to a network ISO image
  • idrac_redfish_storage_controller – Configures the storage controller settings
  • idrac_reset – Reset iDRAC
  • idrac_server_config_profile – Export or Import iDRAC Server Configuration Profile (SCP)
  • idrac_syslog – Enable or disable the syslog on iDRAC
  • idrac_system_info – Get the PowerEdge Server System Inventory
  • idrac_timezone_ntp – Configures time zone and NTP on iDRAC
  • idrac_user – Configure settings for user accounts
  • ome_application_certificate – This module allows to generate a CSR and upload the certificate
  • ome_application_network_address – Updates the network configuration on OpenManage Enterprise
  • ome_application_network_proxy – Updates the proxy configuration on OpenManage Enterprise
  • ome_application_network_time – Updates the network time on OpenManage Enterprise
  • ome_application_network_webserver – Updates the Web server configuration on OpenManage Enterprise
  • ome_device_info – Retrieves the information of devices inventoried by OpenManage Enterprise
  • ome_firmware – Firmware update of PowerEdge devices and its components through OpenManage Enterprise
  • ome_firmware_baseline – Create a firmware baseline on OpenManage Enterprise
  • ome_firmware_baseline_compliance_info – Retrieves baseline compliance details on OpenManage Enterprise
  • ome_firmware_baseline_info – Retrieves baseline details from OpenManage Enterprise
  • ome_firmware_catalog – Creates a catalog on OpenManage Enterprise
  • ome_identity_pool – Manages identity pool settings on OpenManage Enterprise
  • ome_job_info – Get job details for a given job ID or an entire job queue on OpenMange Enterprise
  • ome_network_port_breakout – This module allows to automate the port portioning or port breakout to logical sub ports
  • ome_network_vlan – Create, modify & delete a VLAN
  • ome_network_vlan_info – Retrieves the information about networks VLAN(s) present in OpenManage Enterprise
  • ome_powerstate – Performs the power management operations on OpenManage Enterprise
  • ome_smart_fabric – Create, modify or delete a fabric on OpenManage Enterprise Modular
  • ome_smart_fabric_uplink – Create, modify or delete a uplink for a fabric on OpenManage Enterprise Modular
  • ome_template – Create, modify, deploy, delete, export, import and clone a template on OpenManage Enterprise
  • ome_template_identity_pool – Attach or detach an identity pool to a requested template on OpenManage Enterprise
  • ome_template_info – Retrieves template details from OpenManage Enterprise
  • ome_template_network_vlan – Set tagged and untagged vlans to native network card supported by a template on OpenManage Enterprise
  • ome_user – Create, modify or delete a user on OpenManage Enterprise
  • ome_user_info – Retrieves details of all accounts or a specific account on OpenManage Enterprise
  • redfish_firmware – To perform a component firmware update using the image file available on the local or remote system
  • redfish_powerstate – Manage device power state
  • redfish_storage_volume – Manages the storage volume configuration
