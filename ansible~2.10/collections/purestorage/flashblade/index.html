Plugin Index

These are the plugins in the purestorage.flashblade collection

Modules

  • purefb_alert – Configure Pure Storage FlashBlade alert email settings
  • purefb_banner – Configure Pure Storage FlashBlade GUI and SSH MOTD message
  • purefb_bladename – Configure Pure Storage FlashBlade name
  • purefb_bucket – Manage Object Store Buckets on a Pure Storage FlashBlade.
  • purefb_bucket_replica – Manage bucket replica links between Pure Storage FlashBlades
  • purefb_certgrp – Manage FlashBlade Certifcate Groups
  • purefb_certs – Manage FlashBlade SSL Certifcates
  • purefb_connect – Manage replication connections between two FlashBlades
  • purefb_dns – Configure Pure Storage FlashBlade DNS settings
  • purefb_ds – Configure FlashBlade Directory Service
  • purefb_dsrole – Configure FlashBlade Management Directory Service Roles
  • purefb_fs – Manage filesystemon Pure Storage FlashBlade`
  • purefb_fs_replica – Manage filesystem replica links between Pure Storage FlashBlades
  • purefb_info – Collect information from Pure Storage FlashBlade
  • purefb_inventory – Collect information from Pure Storage FlashBlade
  • purefb_lifecycle – Manage FlashBlade object lifecycles
  • purefb_network – Manage network interfaces in a Pure Storage FlashBlade
  • purefb_ntp – Configure Pure Storage FlashBlade NTP settings
  • purefb_phonehome – Enable or Disable Pure Storage FlashBlade Phone Home
  • purefb_policy – Manage FlashBlade policies
  • purefb_proxy – Configure FlashBlade phonehome HTTPs proxy settings
  • purefb_ra – Enable or Disable Pure Storage FlashBlade Remote Assist
  • purefb_remote_cred – Create, modify and delete FlashBlade object store remote credentials
  • purefb_s3acc – Create or delete FlashBlade Object Store accounts
  • purefb_s3user – Create or delete FlashBlade Object Store account users
  • purefb_smtp – Configure SMTP for Pure Storage FlashBlade
  • purefb_snap – Manage filesystem snapshots on Pure Storage FlashBlades
  • purefb_snmp_agent – Configure the FlashBlade SNMP Agent
  • purefb_snmp_mgr – Configure FlashBlade SNMP Managers
  • purefb_subnet – Manage network subnets in a Pure Storage FlashBlade
  • purefb_syslog – Configure Pure Storage FlashBlade syslog settings
  • purefb_target – Manage remote S3-capable targets for a FlashBlade
  • purefb_user – Modify FlashBlade local user account password
