theforeman.foreman.foreman – Foreman inventory source

Note

This plugin is part of the theforeman.foreman collection.

To install it use: ansible-galaxy collection install theforeman.foreman.

To use it in a playbook, specify: theforeman.foreman.foreman.

  • Synopsis
  • Requirements
  • Parameters
  • Examples

Synopsis

  • Get inventory hosts from Foreman.
  • Uses a YAML configuration file that ends with foreman.(yml|yaml).

Requirements

The below requirements are needed on the local controller node that executes this inventory.

  • requests >= 1.1
