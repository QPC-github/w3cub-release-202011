Class yii\validators\DefaultValueValidator

All Classes | Properties | Methods
Inheritance yii\validators\DefaultValueValidator » yii\validators\Validator » yii\base\Component » yii\base\Object
Implements yii\base\Configurable
Available since version 2.0
Source Code https://github.com/yiisoft/yii2/blob/master/framework/validators/DefaultValueValidator.php

DefaultValueValidator sets the attribute to be the specified default value.

DefaultValueValidator is not really a validator. It is provided mainly to allow specifying attribute default values when they are empty.

This property is overwritten to be false so that this validator will be applied when the value being validated is empty.

public boolean $skipOnEmpty = false

$value public property

The default value or an anonymous function that returns the default value which will be assigned to the attributes being validated if they are empty. The signature of the anonymous function should be as follows,

function($model, $attribute) {
    // compute value
    return $value;
}
