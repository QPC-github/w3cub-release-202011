Class yii\bootstrap\ButtonDropdown

All Classes | Properties | Methods | Events
Inheritance yii\bootstrap\ButtonDropdown » yii\bootstrap\Widget » yii\base\Widget » yii\base\Component » yii\base\Object
Implements yii\base\Configurable, yii\base\ViewContextInterface
Uses Traits yii\bootstrap\BootstrapWidgetTrait
Available since version 2.0
Source Code https://github.com/yiisoft/yii2-bootstrap/blob/master/ButtonDropdown.php

ButtonDropdown renders a group or split button dropdown bootstrap component.

For example,

// a button group using Dropdown widget
echo ButtonDropdown::widget([
    'label' => 'Action',
    'dropdown' => [
        'items' => [
            ['label' => 'DropdownA', 'url' => '/'],
            ['label' => 'DropdownB', 'url' => '#'],
        ],
    ],
]);

See also:

  • http://getbootstrap.com/javascript/#buttons
  • http://getbootstrap.com/components/#btn-dropdowns
