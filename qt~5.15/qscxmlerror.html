QScxmlError Class

The QScxmlError class describes the errors returned by the Qt SCXML state machine when parsing an SCXML file. More...

Header: #include <QScxmlError>
qmake: QT += scxml
Since: Qt 5.7

This class was introduced in Qt 5.7.
