wp_is_recovery_mode()

Is WordPress in Recovery Mode.

Description

In this mode, plugins or themes that cause WSODs will be paused.

Return

(bool)

Source

File: wp-includes/load.php 

function wp_is_recovery_mode() {
	return wp_recovery_mode()->is_active();
}
