get_site_meta( int $site_id, string $key = '', bool $single = false )

Retrieves metadata for a site.

Parameters

$site_id

(int) (Required) Site ID.

$key

(string) (Optional) The meta key to retrieve. By default, returns data for all keys.

Default value: ''

$single

(bool) (Optional) Whether to return a single value. This parameter has no effect if $key is not specified.

Default value: false

Return

(mixed) An array if $single is false. The value of meta data field if $single is true. False for an invalid $site_id.

Source

File: wp-includes/ms-site.php 

function get_site_meta( $site_id, $key = '', $single = false ) {
	return get_metadata( 'blog', $site_id, $key, $single );
}
