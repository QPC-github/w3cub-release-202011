is_author( int|string|int[]|string[] $author = '' )

Determines whether the query is for an existing author archive page.

Description

If the $author parameter is specified, this function will additionally check if the query is for one of the authors specified.

For more information on this and similar theme functions, check out the Conditional Tags article in the Theme Developer Handbook.

Parameters

$author

(int|string|int[]|string[]) (Optional) User ID, nickname, nicename, or array of such to check against.

Default value: ''

Return

(bool) Whether the query is for an existing author archive page.

Source

File: wp-includes/query.php 

function is_author( $author = '' ) {
	global $wp_query;

	if ( ! isset( $wp_query ) ) {
		_doing_it_wrong( __FUNCTION__, __( 'Conditional query tags do not work before the query is run. Before then, they always return false.' ), '3.1.0' );
		return false;
	}

	return $wp_query->is_author( $author );
}
